# kidneed

### Getting Started

1. Clone the project
2. install dependencies with running below command:

```bash
yarn
```

to install yarn use following guide:
https://classic.yarnpkg.com/lang/en/docs/install

3. copy `.env.example` to `.env`

```bash
cp .env.example .env
```

4. run project in development:

```
yarn development
```
